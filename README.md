## Etheasy

## Description
An interactive documentation for Ethers.js

## For development 
1. Install all the dependencies <br/> 
    `yarn` or `yarn install`
2. Start the development server <br/>
    `yarn start`
3. Visit [localhost:3000](http://localhost:3000)
